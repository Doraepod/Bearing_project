-m Минимум необходимых функций:

cd - смена директории, адрес пишется через \
ctrl+insert - копировать выделенное в буфер
shift+insert - вставить из буфера

git add - добавить файлы в репозиторий, использовать с оператором -A
git commit - сделать коммит (применить все изменения), использовать с оператором -m 'сообщение'
git push - залить все на удаленный репозиторий
git pull - стянуть файлы с удаленного репозитория

прокси
git config --global http.proxy http://10.128.0.100:8080
отключение прокси
git config --global --unset http.proxy
